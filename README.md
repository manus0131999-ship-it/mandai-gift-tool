# 万代ギフト配送不可地域検索ツール

このツールは、万代ギフトの配送不可地域を郵便番号または島名で検索できるWebアプリケーションです。

## 使用方法

### 直接アクセス
https://manus0131999-ship-it.github.io/mandai-gift-tool/

### Notionに埋め込む

Notionで以下の手順で埋め込みコードを使用できます：

1. Notionのページを開く
2. 「/embed」と入力して埋め込みブロックを追加
3. 以下のURLを貼り付け：
   ```
   https://manus0131999-ship-it.github.io/mandai-gift-tool/
   ```

## 機能

- 郵便番号での検索
- 島名での検索
- 配送不可地域の詳細情報表示
- リセット機能
